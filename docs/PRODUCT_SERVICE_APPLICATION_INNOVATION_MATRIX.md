# Product Service Application Innovation Matrix

## Purpose
This document defines a matrix for comparing innovation opportunities across products, services, and applications manufactured from the lippytm repo ecosystem.

The goal is to identify which innovation paths create the strongest reuse, growth, and cross-system value.

## Matrix Dimensions
For each innovation opportunity compare:
- product potential
- service potential
- application potential
- cross-repo reuse
- cross-platform fit
- capability growth impact
- review/governance complexity
- portfolio priority

## Best Practices
- favor opportunities that can become more than one type of asset
- identify where one strong innovation can support products, services, and applications simultaneously
- use governance complexity to pace rollout speed
- connect matrix entries back to opportunity IDs and catalog entries
