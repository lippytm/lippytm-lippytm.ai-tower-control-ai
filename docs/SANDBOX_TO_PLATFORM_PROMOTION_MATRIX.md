# Sandbox to Platform Promotion Matrix

## Purpose
This document maps sandbox outputs to best-fit platform surfaces for broader packaging and delivery across the lippytm ecosystem.

## Promotion Targets
- GitHub
- ChatGPT / OpenAI
- Claude
- Grok
- MyClaw
- ManyChat
- BotBuilders
- websites / dashboards
- internal utilities

## Sandbox Output Types
- reasoning packet
- diagnostics packet
- workflow pattern
- multimedia simulation pattern
- database/object pattern
- training walkthrough
- application component
- AI solution package

## Best Practices
- keep GitHub as the authoritative architecture and tracking layer
- use the matrix to choose first-wave promotion targets
- preserve review-level visibility when sandbox outputs move to broader platforms
- prefer outputs with strong repeatability scores for multi-platform promotion
