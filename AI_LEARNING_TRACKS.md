# AI Learning Tracks

This document defines the major AI learning and development tracks across the `lippytm` GitHub ecosystem.

## 1. Conversational AI Track
Primary repos:
- `Chatlippytm.ai.Bots`
- `Clawlippytm.ai.Bots`
- `lippytm.ai`

Focus:
- chatbot identity
- prompt design
- conversation UX
- public bot demos
- product positioning

## 2. Internal AI Operations Track
Primary repos:
- `MyClaw.lippytm.AI-`
- `lippytm-lippytm.ai-tower-control-ai`

Focus:
- agent workflows
- internal task routing
- automation governance
- developer support
- operational safety

## 3. Experimental AI Research Track
Primary repos:
- `OpenClaw-lippytm.AI-`
- `AI-Time-Machines`

Focus:
- sandbox experimentation
- reference architectures
- prototype workflows
- agent research
- speculative system design

## 4. AI + Blockchain Development Track
Primary repos:
- `Web3AI`
- `blockchain-builder-lab`
- `programming-foundations`

Focus:
- GitHub-first development
- applied AI coding
- blockchain systems
- project labs
- portfolio development

## 5. Knowledge and Applied Learning Track
Primary repos:
- `The-Encyclopedia-of-Everything-Applied-ChatAIBots`
- `lippytm.ai`

Focus:
- concept mapping
- educational development
- applied knowledge systems
- cross-domain learning

## 6. Productization Track
Primary repos:
- `Factory.ai`
- `Clawlippytm.ai.Bots`
- `lippytm.ai`

Focus:
- reusable kits
- templates
- deployment patterns
- commercialization pathways

## Guiding Principle
Keep each learning track modular, cross-linked, and connected back to the control tower so experimentation, education, productization, and operations can grow together without becoming confused.
