# Daily Money Actions Workflow

Purpose: create a daily action loop that connects revenue, repo improvements, affiliate promotion, support, and automation.

Quality is Job #1. Keep actions realistic, trackable, and reviewable.

## Trigger

Run daily manually or through a future Twin/ChatGPT automation.

## Inputs

- current leads
- active affiliate links
- current repo priorities
- service packages
- support issues
- money path dashboard
- platform status dashboard

## Output

A daily list with:

1. one lead follow-up
2. one affiliate promotion action
3. one content action
4. one repo improvement action
5. one support improvement action
6. one product/package improvement action
7. one quality/security check
8. one dashboard update

## Daily checklist

- [ ] Review `dashboards/MONEY_PATH_STATUS.md`
- [ ] Pick one offer to promote
- [ ] Draft one follow-up or post
- [ ] Improve one repo file or issue
- [ ] Improve one support asset
- [ ] Improve one product/package asset
- [ ] Update one dashboard

## Review boundary

Drafting is automatic-friendly. Sending, publishing, claiming funding outcomes, or making financial/legal/tax/investment/trading promises requires human review.
