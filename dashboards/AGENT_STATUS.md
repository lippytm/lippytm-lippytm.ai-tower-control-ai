## Agent execution boundaries

### May draft automatically

- GitHub issue drafts
- prompt drafts
- documentation drafts
- support reply drafts
- marketing content drafts
- workflow checklists
- repo improvement suggestions

### Requires human review

- code changes
- deployments
- customer-facing publication
- funding/finance/tax/legal claims
- payment workflows
- API credential setup
- customer data handling
- physical-world/robotics actions
